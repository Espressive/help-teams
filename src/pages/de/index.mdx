## Hier ist eine Liste der Registerkarten, die Sie in <BotName /> for Microsoft Teams finden können:

**Chat:** Hier können Sie mit <BotName /> interagieren, vorzugsweise in kurzen Sätzen, indem Sie das Feld zum Verfassen am unteren Bildschirmrand verwenden. Zum Beispiel können Sie fragen: "Wie teile ich eine Datei in Drive?" oder benachrichtigen Sie "Ich habe ein Problem mit meinem Laptop", und <BotName /> wird eine benutzerdefinierte Antwort unter Verwendung ausgefeilter Verarbeitung natürlicher Sprache (VNS)-Modelle bereitstellen.

Sie können die Unterhaltung mit einer Kombination aus Text, Schaltflächen und anderen von <BotName /> bereitgestellten Steuerungen im Chat fortsetzen. Sie können die Unterhaltung auch jederzeit zurücksetzen, indem Sie Unterhaltung neu starten eingeben oder auf die Schaltfläche Neu starten klicken. Wenn ein Ticket erstellt wird, erhalten Sie eine Benachrichtigung mit einem Link zur Registerkarte Home.

**Home:** Diese Registerkarte zeigt adaptive Karten für jedes Ihrer Tickets, Ankündigungen, die Ihnen von Administratoren gesendet werden, und Genehmigungsanfragen, die von rollenbasierten Zugriffskontrollen abhängen. Die Karten können durch Klicken auf die Option Typ gefiltert werden. Die Karten werden nach der zuletzt aktualisierten zuerst sortiert, die Sie auf die älteste aktualisiert zuerst  ändern können.

Je nach Typ und Lebenszyklusstatus der einzelnen Artikel werden Ihnen für jede Karte unterschiedliche Menüoptionen angezeigt.

Mit der Schaltfläche Kommentar können Sie Details zu Ihren Anfragen hinzufügen, einschließlich Anhängen.

Sie können ein Ticket stornieren, indem Sie im Menü auf Stornieren klicken. Karten auf der Registerkarte Home können auch Links zu Fremdsystemen enthalten.

Auf der Registerkarte Home finden Sie auch von Administratoren gesendete Ankündigungen und Genehmigungsanfragen, die von Ihren rollenbasierten Zugriffskontrollen abhängen.

## Hier ist eine Liste mit Produktivitätstipps zur Verwendung von <BotName />.

### Anheften <BotName />

<BotName /> kann an die App-Leiste geheftet werden, so dass es immer einen Klick entfernt ist.

Wenn Sie es nicht in Ihrer App-Leiste sehen können:

1. Klicken Sie auf die Schaltfläche Weitere Apps (...) in der App-Leiste (linkes Menü auf Ihrem Bildschirm).
2. Wählen Sie <BotName /> aus den zuletzt verwendeten Apps oder suchen Sie, indem Sie "<BotName />" in die Suchleiste eingeben.
3. Klicken Sie mit der rechten Maustaste auf die <BotName />-App und wählen Sie Anheften aus dem Kontextmenü.

### Freigeben <BotName />

Microsoft Teams gibt Benutzern die Möglichkeit, Apps zu "blockieren" oder vorübergehend zu deaktivieren.

Wenn Sie auf der Registerkarte Chat eine Nachricht sehen, die besagt, dass "Sie keine Nachrichten an diesen Bot senden können, weil Sie ihn blockiert haben", können Sie den Zugriff wiederherstellen, indem Sie mit der rechten Maustaste auf die <BotName />-App klicken und im Kontextmenü die Option Freigeben auswählen.
